# Admin Dashboard Front-end

Production-ready Admin Dashboard built with React + Vite, focused on real-world management workflows.

## Features

- Authentication flow (fake login with `localStorage`)
- Dashboard overview cards, status chart, and recent orders table
- Users management: search, filter, pagination, add/edit/delete
- Products management: search, category filter, price sort, pagination, add/edit/delete
- Orders management: status filter and inline status update
- Reusable loading, empty, and error states
- Responsive layout with fixed sidebar and sticky header

## Tech Stack

- React (Vite)
- JavaScript ES6+
- React Router
- TailwindCSS
- Axios (service layer)
- localStorage (mock database + auth)

## Project Structure

```text
src/
  components/
  hooks/
  layouts/
  pages/
  services/
  utils/
```

## Run Locally

```bash
npm install
npm run dev
```

## Build
